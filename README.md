# Java DSA Questions & Solutions

Welcome to the **DSA Questions & Solutions** repository! This repository contains a curated collection of Java implementations for various Data Structures and Algorithms (DSA) concepts, ranging from basic array manipulations to common algorithm patterns.

## 📂 Repository Contents

Below is a summary of the Java programs included in this repository:

| File Name | Description |
| :--- | :--- |
| [`InputOutput.java`](./InputOutput.java) | Demonstrates standard input and output operations in Java. |
| [`MaximumElement.java`](./MaximumElement.java) | Finds the maximum element in an array. |
| [`MinimumElement.java`](./MinimumElement.java) | Finds the minimum element in an array. |
| [`NegativeElementOnly.java`](./NegativeElementOnly.java) | Processes or filters negative elements within an array. |
| [`OddEvenProblem.java`](./OddEvenProblem.java) | Separates or handles odd and even numbers. |
| [`PassingEleToMethods.java`](./PassingEleToMethods.java) | Illustrates how to pass array elements or references to methods. |
| [`ProductOfArray.java`](./ProductOfArray.java) | Calculates the product of elements in an array. |
| [`ShallowDeepCopy.java`](./ShallowDeepCopy.java) | Demonstrates the difference between shallow copy (reference assignment) and deep copy (cloning/copying values) using `Arrays.copyOf`. |
| [`Sort.java`](./Sort.java) | Implements/uses array sorting techniques. |
| [`SumOfElements.java`](./SumOfElements.java) | Calculates the sum of all elements in an array. |
| [`TwoSum.java`](./TwoSum.java) | Implements the classic Two Sum problem using arrays. |

## 🚀 How to Run Locally

To compile and run any of these files on your local machine:

1. **Ensure you have Java JDK installed** (version 8 or higher is recommended).
2. **Compile the program:**
   ```bash
   javac FileName.java
   ```
3. **Run the program:**
   ```bash
   java FileName
   ```
